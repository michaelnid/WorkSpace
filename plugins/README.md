# Plugins-Verzeichnis

Dieses Verzeichnis ist für aktive/laufende Plugins reserviert.

- Offizielles Beispiel-Plugin: `PLUGIN_DEV/beispiel-plugin/`
- Template-/Lernmaterial liegt im `PLUGIN_DEV`-Ordner, nicht hier.
- Plugin-Ordner mit führendem Unterstrich (`_`) werden vom Loader und Build ignoriert.
